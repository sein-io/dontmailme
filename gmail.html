Gmail Setup

Takes about 3 minutes. Runs completely within your own Google account.

1 Create a new Script

Go to script.google.com and click New Project.

2 Enable Gmail API

Click the Services (+) icon. Select Gmail API and click Add.

3 Personalize & Copy Code

Add email addresses you want to keep (whitelist):

4 Test safely

Select testDryRun and click Run. Check the logs to see what would happen.

5 Activate

Select installTrigger and click Run.
