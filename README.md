# DontMailMe.org

> Stop the noise and plant trees.

DontMailMe is a privacy-first, zero-data-access script that automatically unsubscribes you from newsletters via RFC 8058.

## 🚀 Key Features
* **Zero Data Access:** Runs locally in your Google Apps Script environment.
* **Strict RFC 8058:** Uses `List-Unsubscribe-Post` for secure one-click unsubs.
* **Eco-Friendly:** Every unsubscribed newsletter reduces server energy.
